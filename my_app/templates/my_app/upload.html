Best file formats for the MVP

The analyzer works best when the uploaded file has a clear tabular structure.

Excel (.xlsx)

Best option for clean operator reports and manual corrections before upload.

CSV (.csv)

Fast and lightweight option for already cleaned tabular data.

Excel (.xls)

Older spreadsheet format that can still be used if the structure is correct.

XML (.xml)

May work if the XML is in a clear tabular form, though Excel is preferred.

What the system will do

Processing pipeline after upload

  • 1 Read the uploaded file and detect available columns.
  • 2 Let the user map the time and duration columns manually.
  • 3 Convert time into 4 intervals: 00-06, 06-12, 12-18, 18-24.
  • 4 Build joint frequency and probability tables from categorized variables.
  • 5 Compute mean, variance, standard deviation, covariance, and correlation.
  • 6 Show charts and a final automatic conclusion in the result dashboard.
MVP Goal

Why this module comes first

The Call Log Statistical Analyzer is the strongest demonstrable module for the first version because it combines file handling, real data transformation, statistical computation, chart generation, and dashboard presentation in one complete flow.

Once this module is stable, the remaining modules — Distribution Calculator, Problem Solver, and Topic-Based Trainer — can be added on top of the same project structure.
