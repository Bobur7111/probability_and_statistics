Main MVP Focus

Call Log Statistical Analyzer workflow

The MVP is centered around one complete analytical flow: file upload, variable mapping, transformation, statistical computation, visualization, and automatic conclusion. This makes the project highly demonstrable even while other modules continue expanding.

1

Upload raw call log file

The user uploads a structured Excel or CSV file exported from mobile operators. The platform stores the file temporarily and prepares it for processing.

2

Map the key analytical columns

The user selects which column contains the call time and which column contains the call duration or used amount. This makes the module flexible for different report formats.

3

Transform data into categories

Time is converted into four intervals of the day, while duration is grouped into ranges such as 0-60, 60-300, 300-600, and 600+.

4

Build statistical tables and graphs

The system generates joint frequency and probability distributions, marginal distributions, charts, and a summary interpretation.

5

Compute analytical indicators

Expected value, variance, standard deviation, covariance, and correlation are computed using coded categories of X and Y.

6

Show conclusion and final report

The final dashboard presents visual analytics and automatic conclusions, while future versions can support downloadable report export.

Core Statistics

Main indicators used in the analyzer

The current MVP is centered on joint distributions and classical statistical measures for coded random variables derived from call time intervals and duration categories.

Joint Frequency Table
nij
Counts how many observations fall into each pair of time and duration categories.
Joint Probability
pij = nij / n
Transforms raw counts into probabilities based on the total number of valid observations.
Expected Value
M[X], M[Y]
Represents the average coded position of the random variables X and Y.
Variance
D[X], D[Y]
Measures how widely the random variables are distributed around their expected values.
Standard Deviation
σ(X), σ(Y)
Shows the spread of categories in the same coded scale as the variables.
Covariance
Cov(X,Y)
Describes how the coded time and duration categories vary together.
Correlation
r(X,Y)
Shows the direction and strength of the relationship between the variables.
Marginal Distributions
P(X), P(Y)
Summarize probabilities for each variable separately from the joint distribution.
